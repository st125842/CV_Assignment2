## 📁 File Descriptions

- **F1_CUT** — Input video used for object tracking.
- **KCF_CUT** — Output video showing the tracking result using the **KCF (Kernelized Correlation Filters)** algorithm.
- **CSRT_CUT** — Output video showing the tracking result using the **CSRT (Discriminative Correlation Filter with Channel and Spatial Reliability)** algorithm.
